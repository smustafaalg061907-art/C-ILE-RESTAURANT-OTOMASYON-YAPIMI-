# C-ILE-RESTAURANT-OTOMASYON-YAPIMI-
C# WinForms ve .NET ile geliştirilen bu otomasyon; modern, kutucuk tabanlı arayüzüyle restoran yönetimini tek noktada toplar. Dinamik masa ve rezervasyon takibi, paket servis , anlık kasa raporlama ve daha fazla özellik içerir. OOP prensipleriyle kurgulanan yapısı, yetkilendirme sistemi ve şık tasarımıyla operasyonel verimliliği artırır.
